┌───────────────────────────────────────────────────────────────┐
│  PRÉ-REQUISITO: Instalar Node.js                              │
└───────────────────────────────────────────────────────────────┘

1. Baixe Node.js em: https://nodejs.org
2. Instale (versão LTS recomendada)
3. Verifique: abra terminal e digite "node --version"


┌───────────────────────────────────────────────────────────────┐
│  PASSO 1: Instalar Dependências                               │
└───────────────────────────────────────────────────────────────┘

Abra o terminal na pasta "ouvidoria-backend" e execute:

    npm install

Aguarde o download das dependências (~30 segundos)


┌───────────────────────────────────────────────────────────────┐
│  PASSO 2: Iniciar o Servidor                                  │
└───────────────────────────────────────────────────────────────┘

No mesmo terminal, execute:

    npm start

Você verá uma mensagem de confirmação com:
🚀 SERVIDOR OUVIDORIA DIGITAL INICIADO!
📍 Servidor rodando em: http://localhost:3000


┌───────────────────────────────────────────────────────────────┐
│  PASSO 3: Acessar o Sistema                                   │
└───────────────────────────────────────────────────────────────┘

Abra seu navegador (Chrome, Firefox, Edge) e acesse:

    http://localhost:3000

PRONTO! O sistema está funcionando! 🎉

🐛 PROBLEMAS COMUNS:

❌ "Cannot find module"
   Solução: Execute "npm install"

❌ "Port 3000 already in use"
   Solução: Mude a porta em server.js (linha 6)
   ou mate o processo usando a porta

❌ "Node não é reconhecido"
   Solução: Instale Node.js de nodejs.org

❌ Página não carrega
   Solução: Verifique se o servidor está rodando
   no terminal
